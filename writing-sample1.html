How to Protect your Software from Illegal Use?

Illegal use can include stealing your code through reverse engineering or could be as simple as using multiple copies of the software that were not purchased.
There are two main strategies used to protect software from illegal use:

1. Reverse Engineering Protection - Tools can be used to copy software code from memory and use it illegally. CrypKey's Stealth feature encrypts the code and breaks it into thousands of pieces while running in memory, making it virtually impossible to reverse engineer.

2. Hardware Locking - the biggest leakage of revenue is from extra copies of software being used that were not paid for. To protect against this, locking a licensing key to the computer's hardware is the most effective method. Since you control the authorization keys, you control how many copies of your software are used. CrypKey uses a sophisticated 'intelligent hardware system' to look at several hardware identifiers on the computer, and creates a secure locking code. This allows a small tolerance for hardware changes, but protects your software from being copied to other computers without your authorization. Another option that provides more mobility is locking it to a USB Flash Drive. CrypKey USBKeys can be used with CrypKey Instant or SDK and allows your customer to use the software on any Windows computer. However, they are restricted to using only one computer at a time.
